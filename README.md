# Student Management System (SQL Beginner Project)

## Project Overview
This is a beginner SQL project built using MySQL. It manages students, courses, and enrollments using a relational database.

## Database Tables
- Students
- Course
- Enrollments

## SQL Concepts Used
- CREATE DATABASE
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- INNER JOIN
- COUNT
- SUM
- AVG
- MIN
- MAX
- UPDATE
- DELETE

## Project Structure
- create_tables.sql
- insert_data.sql
- queries.sql
## Total Records
- Students: 20
- Courses: 5
- Enrollments: 30

## Tools Used
- MySQL
- MySQL Workbench

This project helped me practice fundamental SQL concepts by solving real-world business queries.
